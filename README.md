# Blog

A portfolio project

# How to

1. Install [Docker](http://docker.com)

2. Install [Node.js](https://nodejs.org)

3. Clone this repository

```sh
git clone git@github.com:brtmvdl/blog.git
cd blog
```

4. Run the project

```sh
bash ./env/run.sh 
```

5. Open [0.0.0.0:3000/blog](http://0.0.0.0:3000/blog) in browser
